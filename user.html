ব্যবহার গাইড / User Guide

কীভাবে ডাউনলোড করবেন?

১. পছন্দের টেম্পলেট কার্ডে ক্লিক করুন
২. "ডাউনলোড" বাটনে ক্লিক করুন
৩. ফাইলটি আপনার ডিভাইসে সেভ হবে
৪. ZIP ফাইল হলে Extract করে ব্যবহার করুন

কাস্টমাইজ করবেন কীভাবে?

HTML/CSS: VS Code বা Notepad++ দিয়ে সম্পাদনা করুন
Figma: Figma.com এ Import করুন (Free account যথেষ্ট)
PSD: Photoshop বা GIMP (বিনামূল্যে) দিয়ে সম্পাদনা করুন
Word: MS Word বা Google Docs দিয়ে সম্পাদনা করুন

লাইসেন্স / License

সকল ফ্রি টেম্পলেট ব্যক্তিগত ও বাণিজ্যিক উভয় কাজে ব্যবহার করতে পারবেন। ক্রেডিট দেওয়া বাধ্যতামূলক নয়, তবে প্রশংসিত হবে।

সাহায্য / Help

কমিউনিটিতে প্রশ্ন করুন বা টেম্পলেট রিকোয়েস্ট করুন। আমাদের টিম ২৪ ঘণ্টার মধ্যে সাড়া দেয়।
